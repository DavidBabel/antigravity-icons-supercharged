A fork from [miguelsolorio/vscode-symbols](https://github.com/miguelsolorio/vscode-symbols), main icons used in **[Antigravity](https://antigravity.google/)** (the fork of [vscode](https://github.com/microsoft/vscode) made by [Google](https://github.com/google)).

_Mainly created for personal use. If an icon is missing, feel free to [contribute](#contributing), I made it very easy to do._

Premium display of Angular files.

⭐ Check this amazing extension also : [clever-vscode](https://marketplace.visualstudio.com/items?itemName=davidbabel.clever) ⭐
Trust me, you might love it.
